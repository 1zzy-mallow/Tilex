Events

    
when starting
when (any v) key pressed
when cursor clicked
when cursor moved
when cursor touching tile > ()
    

    

      when starting: when the code is executed

      when (any v) key pressed: runs when the selected key is pressed and all keys on the keyboard can be used

      when cursor clicked: runs when the user clicks anywhere

      when cursor moved: runs every time the mouse moves

      when cursor touching tile > (): runs when the cursor touches a tile greater than the given value
    

  


  

    
Control

    
if <>...
//something
fi

switch ()*
case ()...
//something
esac
end

while <>...
//something
stick

repeat until <>...
//something
litnu
    

    

      if <>...: runs if condition is true

      fi: closing of if block

      switch ()*: monitors the chosen variable

      case ()...: executes if the switched variable equals the case value

      esac: closing of case

      end: closing of switch

      while <>...: runs while the condition is true

      stick: closing of while

      repeat until <>...: repeats until the boolean is false

      litnu: closing of repeat until

      * all switch conditions must contain a variable

      ** all case () conditions must be inside a switch () condition
    

  


  

    
Operators

    
1
0
<()=()>
(()+())
(()-())
(()*())
(()/())
() ? ()
    

    

      1: true

      0: false

      <()=()>: returns true if both values equal the same

      (() + ()): returns the two values combined

      (() - ()): returns the first value minus the second

      (() * ()): returns the first value multiplied by the second

      (() / ()): returns the first value divided by the second

      () ? (): reports a number in between the first and the second
    

  


  

    
Visuals

    
+
-
br+
brtl+
br-
brtl-
change x by ()
change y by ()
draw hex #(________)
    

    

      +: increments value of current tile by 1

      -: decrements value of current tile by 1

      br+: changes brightness of self by 10%

      brtl+: changes brightness of current tile by 10%

      br-: changes brightness of self by -10%

      brtl-: changes brightness of current tile by -10%

      change x by (): changes the x position by the value

      change y by (): changes the y position by the value

      draw hex #(________): leaves a trail behind the user colored the hex color inputted with transparency

Jars

    
[create jar]

jar
<[jar v] contains ()?>
add () to [jar v]
delete () of [jar v]
delete all of [jar v]
when [jar v] contains ()
    

    

      [create jar]: creates a jar

      jar: contains multiple values

      <[jar v] contains ()?>: checks if the jar contains a value

      add () to [jar v]: adds to the chosen jar

      delete () of [jar v]: deletes of the chosen jar

      delete all of [jar v]: deletes all of the chosen jar

      when [jar v] contains (): runs when the chosen jar contains a value
    

  


  

    
Honey (Miscellaneous)

    
[create honey]
define {honey}
honey
display ()
display () for () seconds
wait ()
clear
reload
load site ()
load site nt()
...()
ask () wait
(answer)
marco
polo ()
polo <>
    

    

      [create honey]: make a honey with booleans, reporters and labels plus also run without screen refresh

      define {honey}: runs when the honey is run

      honey: executes the define

      display (): displays text until stop pressed

      display () for () seconds: displays text for seconds

      wait (): waits for seconds

      clear: clears all

      reload: reloads the page

      load site (): loads the inputted site in current page

      load site nt(): loads the inputted site in a new tab

      ...(): random string of text that is long

      ask () wait: asks and waits until an answer is confirmed

      (answer): reports the given answer

      marco: start of a return

      polo (): end of marco and returns a reporter value

      polo <>: end of marco and returns a boolean value
